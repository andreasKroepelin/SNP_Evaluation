SNP Evaluation - Quick Introduction

Input Data

Load a SNP table: Data > Load SNP Table
Load one or two VCF file(s): Data > Load 1 VCF File or Data > Load 2 VCF Files
Optionally, load a protein table: Data > Load Protein Table

Choose SNPs

Click Add tab and then choose whether you want the unique, the shared or all SNPs of a given sample (should be the sample of your VCF file), or the complete SNP table or certain positions which you can specify in a file (one line for each position number).

Configure measures

Click Measures > Configure Measures to open a configuration form. There you can define a new measure by:
  • Single Measure: this value is taken from all genome positions in the specified window
  • Window Measure: this operation produces one number for the whole window
  • Window Size: this number defines how many genome positions around the SNP should be considered
  • Standardization:
    • None: do nothing
    • Subtract Mean: subtract the genome wide mean of the Single Measure
    • Z-Standardization: subtract the genome wide mean and divide by the standard deviation
You then have to give the measure a unique name, e. g. by clicking Use name suggestion.

Evaluate

When you click Evaluate!, you get a table and a graphical representation of the evaluation. For each SNP you find its position, in how many samples it occurs, the values of each of the defined measures and additionally informations derived from the protein table if loaded.
Double click on a row in the table to get to the corresponding row in the bar plot and vice versa.

Further Features

Filtering

At Measures > Set Filters you can open a form where you can specify filters. Each filter regards to one table column and you can use the typical SQL operators.

Comparisons

This is only relevant when loading two VCF files. At Measures > Set Comparisons you can define whether you want to see the ratio or difference of your measures of the first and second VCF file.

Correlations

At Evaluation > View Correlations you can open a window where you can analyse correlations between any two numerical columns of the table. You must specify which table to use for the calculation and which table(s) should appear in the dot plot.

Export

Finally you can export all tables and bar plots at Evaluation > Export evaluation after giving them a name.
